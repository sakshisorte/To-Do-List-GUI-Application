# 📋 To-Do List GUI Application

A modern and user-friendly **To-Do List Desktop Application** developed using **Python** and **Tkinter**. This application helps users efficiently manage daily tasks through a simple graphical interface.

## 📖 Project Overview

The To-Do List GUI Application is a desktop-based task management system that allows users to organize their daily activities. Users can add tasks, mark them as completed, delete tasks, and clear the entire task list through an intuitive graphical interface.

This project demonstrates the use of Python's Tkinter library for developing GUI-based desktop applications.

---

## ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete selected tasks
* 🧹 Clear all tasks
* 📜 Scrollable task list
* 🎨 Simple and user-friendly interface
* ⚠️ Input validation and error handling
